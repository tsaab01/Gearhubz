About HEX and RGB
  • HEX: Six-digit hexadecimal format (#RRGGBB) used in web design
  • RGB: Red, Green, Blue values (0-255) representing color intensity
  • Both represent the same colors, just in different formats
  • HEX is more compact, RGB is more intuitive for calculations
